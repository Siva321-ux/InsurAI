🚀 Next-gen InsurTech platform leveraging AI to automate claims and optimize risk assessment.

---

## 🧠 Overview

InsurAI is an intelligent insurance platform designed to modernize traditional insurance workflows. It uses artificial intelligence to enhance risk evaluation, detect fraudulent claims, and automate decision-making processes in real time.

---

## ❗ Problem

The current insurance ecosystem faces multiple inefficiencies:

* Manual and slow claim processing
* High risk of fraud
* Lack of dynamic and accurate risk assessment
* Poor user experience due to delays

---

## 💡 Solution

InsurAI introduces a smart, automated system that:

* Predicts customer risk scores using machine learning
* Detects fraudulent claims through anomaly detection
* Automates claim approval workflows
* Enables dynamic premium pricing based on user behavior

---

## 🏗️ Architecture

The system follows a modular architecture:

* **Frontend:** React Native (cross-platform mobile app)
* **Backend:** Node.js + Express (REST APIs)
* **Database:** MongoDB
* **ML Layer:** Python (risk scoring + fraud detection models)

📌 Architecture diagram available in `/docs`

---

## 🛠️ Tech Stack

* React Native
* Node.js
* Express.js
* MongoDB
* Python
* TensorFlow / Scikit-learn
* REST APIs

## ⚙️ Setup Instructions

### 1. Clone the Repository

git clone  https://github.com/Siva321-ux/InsurAI

---

### 2. Backend Setup

cd server
npm install
npm start

---

### 3. Frontend Setup

cd client
npm install
npm start

---

### 4. ML Model (Optional)

cd ml
pip install -r requirements.txt
python model.py
